# カウントアップタイマー

シンプルなカウントアップタイマーです。スタート/ストップ/リセット機能付き。


## 機能

- ⏱️ スタートボタンでカウント開始（0秒から）
- ⏸️ ストップボタンでカウント停止
- 🔄 リセットボタンで00:00:00に戻る
- 🕐 時:分:秒 形式で表示（100時間以上にも対応）

## 使用技術

- HTML
- CSS
- JavaScript
  - `setInterval` / `clearInterval` でタイマー制御
  - 秒数を時:分:秒に変換するアルゴリズム

## 学んだこと

- setIntervalの基本的な使い方
- タイマーの開始/停止/リセットの実装方法
- 秒数を時間表示に変換する処理
